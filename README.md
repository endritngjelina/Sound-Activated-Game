# 🎵 Sound-Activated Game

A collection of interactive browser-based games that respond to sound input through your microphone. Built entirely with HTML, CSS, and JavaScript — no installation required.

***

## 🎮 Overview

Sound-Activated Game is a series of 12 progressively developed HTML5 games where player actions and game mechanics are driven by real-time audio input from your microphone. Speak, clap, or make noise to control the game — your voice is the controller.

Each version (`Sound Activated Game 1.html` → `Sound Activated Game 12.html`) represents an iteration of the game with new mechanics, improvements, and features added along the way.

***

## 🚀 Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari)
- A working microphone
- Microphone permission granted in your browser

### How to Play

1. Clone or download this repository:
   ```bash
   git clone https://github.com/endritngjelina/Sound-Activated-Game.git
   ```
2. Open any `.html` file in your browser (start with the latest version for the best experience):
   ```
   Sound Activated Game 12.html
   ```
3. Grant microphone access when prompted.
4. Make noise — clap, speak, or blow — to interact with the game!

> **Tip:** Start with `Sound Activated Game 1.html` to see how the project evolved, or jump straight to `Sound Activated Game 12.html` for the most polished experience.
